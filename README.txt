VVS Authority Pro Admin OBS Game

Open:
1. Unzip the folder.
2. Double-click index.html.

Admin:
- Click Admin.
- Default password: vvs123.
- Click Prize Editor to change prizes, quantities, color tiers, and unlock percentages.
- Save & Build Game reshuffles the board.

Views:
- Audience: clean normal view.
- Admin: private view showing the hidden prize assigned to each spot and unlock percentage.
- OBS Clean: removes side image and controls for a cleaner stream layout.

Unlock rules:
- A prize with an unlock percentage will not reveal until the board reaches that percentage complete.
- If selected early, the app automatically swaps it with an unlocked prize and keeps the locked prize in the game.
- This is meant to be used with clear game rules.

Features:
- 300 spots by default.
- Buyer name.
- Random spot.
- Undo last draw.
- Export history CSV.
- Inventory counter.
- Banger alert for mega prizes.
- Your uploaded VVS Authority image included.

OBS setup:
- Add Browser Source.
- Check Local file.
- Select index.html.
- Set width 1920 and height 1080.
